C programming


1. Working with a Function

a. Define Function

A function is a self-contained block of code designed to perform a specific task. It is used to divide a program into smaller, manageable, and reusable sections, enhancing code organization, readability, and modularity.

b. Syntax of Function

The general structure of a function includes the return type, function name, parameters (if any), and a body containing executable code.
Syntax: return_type function_name(parameter_list) { /* statements */ }

c. Types of Functions

  • Library Functions: Built-in, pre-defined functions provided by the programming language or standard library. They perform common tasks such as input/output, mathematical operations, etc.
  • User-Defined Functions: Functions created by the programmer to perform specific tasks required by the program.

d. Components of a Function

  1. Function Prototype: A declaration that informs the compiler about the function’s name, return type, and parameters before its actual definition.
  2. Function Call: The statement used to invoke or execute a function.
  3. Function Definition: The part where the actual task of the function is written — includes the code that gets executed when the function is called.
  4. Return Type: Specifies the type of value the function will return to the calling part of the program (e.g., int, float, void).

2. Categories of Functions with Definitions

i. Function with Return Type but No Arguments

This type of function returns a value to the calling function but does not take any input. It performs a task internally and sends back a result.

ii. Function with Return Type and Arguments

This type of function both accepts input (arguments) and returns a value after processing those inputs. It is useful when a function needs external values to perform a task.

iii. Function with Arguments but No Return Type

This function receives input values (arguments) but does not return any result. It simply performs the required operations internally.

iv. Function with No Arguments and No Return Type

This function neither takes any input nor returns any value. It performs an operation independently within its body.
