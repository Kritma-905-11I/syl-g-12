Introduction

Computer is an electronic device device derived from the Latin word "computere" which means to calculate. It accepts raw facts and figure s as an input which are isolated and uninterpreted through input device, process it according to the requirement of the user or command supplied by the user, store it before and after processing of require and produce a meaningful information as an output through an output device. Generally, computer works on IPO (Input-Process-Output) cycle. It follows GIGO (Garbage In Garbage Out) algorithm which means the output is always based on the input supplied by the user. Generations of computer.

History of computers dates back to mechanical era where computer were not used for general purpose. It takes long period of time to come to this phase. Development of computer had passed through different era from mechanical to electro mechanical to digital computer. The development in different electronic components, precise engineering and hi-tech technology has change the operation of computer drastically. Earlier computer were only used for specific purpose such as counting and performing basic mathematical operation. Later on, due to the necessity and requirement of user different general purpose computer were developed. Therefore, the classification of computer on the basis of major electronic component, their architecture and modes of operation is known as generations of computer.
Five different generations of computer are listed below along with their distinct features.

  1. First Generation Computers (1946-1958)

    • They used vacuum tubes as their basic electronic components to control and amplify electronic signals.
    • They were big in size, consumed more electricity, and generated a large amount of heat, thus requiring an air conditioning system to keep the device cool.
    • They were unreliable and prone to frequent hardware failures.
    • They were normally not used for commercial purposes.
    • Machine-level language was used to program these computers.
    • Examples: ENIAC, EDVAC, EDSAC, UNIVAC

  2. Second Generation Computers (1959-1964)

    • They used transistors as their basic electronic components.
    • They were smaller in size compared to the first generation, consumed less electricity, and generated less heat.
    • They were less prone to hardware failure, making them more reliable.
    • They were still not widely used for commercial purposes due to difficult and costly production.
    • Assembly language was used to program these computers.
    • Examples: IBM 7000, IBM 1401, IBM 1620, NCR 304, MARK III, ATLAS
    • Note: The first computer brought to Nepal was IBM 1401 for the population census of 2028 BS.

  3. Third Generation Computers (1965-1974)

    • They used Integrated Circuits (ICs) as their basic electronic components.
    • They were smaller in size, consumed less electricity, and generated less heat.
    • They were faster and more reliable compared to the first and second generations.
    • They were widely used for commercial applications worldwide.
    • High-level languages were used to program these computers.
    • Examples: PDP-8, PDP-11, ICL 2900 series, IBM 360, IBM 370

  4. Fourth Generation Computers (1975-1990)

    • They used VLSI (Very Large Scale Integrated Circuits) as their main electronic components.
    • Their size was drastically reduced, making them much more portable.
    • They were faster in processing and much more reliable.
    • They had larger memory capacities, up to gigabytes (GB) and terabytes (TB).
    • They used advanced high-level programming languages.
    • Examples: Apple Macintosh, IBM PC

  5. Fifth Generation Computers (1990-onwards)

    • They use ULSIC (Ultra Large Scale Integrated Circuit) or Microprocessor as their major electronic components.
    • They incorporate or will incorporate artificial intelligence.
    • They have tremendously high processing power due to parallel processing.
    • They are capable of image processing, speech recognition, and voice recognition.
    • They use various high-level programming languages.

Introduction to Computer System

The word system refers to the integration of several independent units that work together to achieve a common objective or goal. A system is a set of detailed methods, procedures, and routines created to carry out a specific activity, perform a duty, or solve a problem. In a system, different units coordinate with each other to work as a whole. Without any of the units, the entire system is affected.

Similarly, a computer system is also a combination of hardware, software, and peripherals that are organized together to run our computer smoothly. Our computer system operates based on the IPO (Input-Process-Output) cycle. It is mandatory for different units to work together to convert raw facts and figures into meaningful information. Without any one of these units, the computer starts malfunctioning.

The design, arrangement, construction, or organization of different parts of a computer system is known as computer architecture. It is the conceptual design and fundamental operational structure of a computer system. It is a framework that focuses on how the CPU performs internal operations. So, computer architecture can also be defined as the science and art of selecting and interconnecting hardware components to create a computer that meets functional, performance, and cost requirements.

Components/Elements of a Computer System

A computer system has several basic components or elements, which are listed below:

  • Hardware (Input, Output, Processing, and Storage)
  • Software
  • Data/Information
  • Procedure
  • People
  • Communication

1. Hardware

Hardware consists of the physical devices of a computer that can be touched and seen. Without hardware, software cannot function. It includes input devices, output devices, processing units, and storage devices. Examples of hardware devices include keyboards, mice, CPUs, hard disks, and RAM.

2. Software

Software is a collection of instructions known as a program, which performs specific functions. These programs are integrated to form software. Software defines how hardware should function and facilitates proper utilization and mobilization of computer hardware. Examples include operating systems, MS Office, and billing software.

3. Data/Information

Data refers to raw facts and figures that are isolated and uninterpreted. Data is fed into the computer through input devices for processing. The meaningful result obtained after processing data according to supplied instructions is known as information. Information is the processed version of data obtained as output through an output device.

4. People/User

People are the most important part of the computer system. They operate the hardware and create software. Users generate commands to operate the computer system. Without users, no hardware or software can be utilized.

5. Procedure

Procedures define how things are done. They include steps for converting raw facts and figures into meaningful information and general guidelines on how computers should be used.

6. Communication

Once the computer system is set up, there is a continuous flow of data between users and the computer. Data is shared among different components of the computer system in the form of electronic signals. The exchange of data and information between several components is known as communication.

Basic Hardware Components of a Computer System

  • Input Unit
  • Output Unit
  • Processing Unit
  • Storage Unit

Input Unit

Data and instructions must enter the computer system before any computation can be performed. An input device accepts data and instructions from the user. It also converts input data into a suitable form that the computer system can process. Functions of an input unit include:

  • Accepting instructions and data from external sources.
  • Converting supplied instructions and data into a computer-acceptable form (binary code).
  • Supplying the converted instructions and data to the computer system for further processing.

Commonly used input devices: Keyboard, trackpad, touchscreen, mouse, light pen, scanner, BCR, MICR, OCR, joystick, trackball, etc.

Output Unit

The output unit performs the reverse function of the input unit. It provides processed information and results to users. Functions of an output unit include:

  • Accepting results produced by the computer.
  • Converting results into a human-understandable form.
  • Supplying converted results to users.

Commonly used output devices: Monitor, printers, speakers, plotters.

Types of Output Devices:

  1. Hard Copy Output
  2. Soft Copy Output

Hard Copy Output

Hard copy output is printed on paper or other materials, making it permanent and portable. Unlike soft copy output, it does not require continuous power to be viewed. Examples include printed documents from printers or plotters.

Common hard copy output devices: Printers, plotters.

Soft Copy Output

Soft copy output appears on display screens or is produced as sound from speakers. It is temporary, non-portable, and requires continuous electrical supply. Examples include images on a monitor or audio from speakers.

Common soft copy output devices: Monitors, speakers.

Central Processing Unit

Central Processing Unit

The CPU is brain of any computer system. Like in human body, all major decision are taken by the brain. Similarly, in computer all major calculation and comparison are made inside the CPU and CPU is also responsible for activating and controlling the operation of other unit of a computer system. A micro processor is a single chip that performs the function of CPU in micro-computers. The speed of the microprocessor is measured in terms of Megahertz and Gigahertz.

The functions of CPU (processor) are:

  • To carry out processing.
  • To give command and co-ordinate with all other parts of computer system.
  • To control the sequence of operation that takes place in our computer system.
  • To helps in memory allocation and controls storage of data and instruction.

The central processing unit consists of unit:

  • Arithmetic logic Unit (ALU)
  • Control Unit (CU)
  • Memory Unit (MU)

ALU: Arithmetic Logic Unit

ALU stands for Arithmetic Logic Unit. It is one of the most important unit of processing. Where actual execution of the instruction takes place during processing. It performs arithmetic and logical operations and controls speed of those operations as directed by control unit(CU). To be clear all the calculation and decision are made in ALU. The data and instruction are stored in the primary memory before processing, and are transferred when needed to the ALU.

CU: Control Unit

Although, control unit doesn't perform any actual processing on the data , it acts as a central nervous system for other component of the computer. It manages and co-ordinate the entire computer system. It obtain instruction from the program stored in the main memory, interprets the instruction accordingly and issue signal that cause other unit of the system to execute them. Basically, it controls overall operation of the computer. It also controls all other device input/output device connected to the CPU. It directs the movement of electrical signal between main memory to ALU and input-output device. Hence, control unit acts as the nerve system of the computer system.

MU: Memory Unit

Memory unit is responsible for storing data and instruction either for short or longer period of time. Basically memory unit stores data and instruction before and after processing . As we know, memory are needed for program being currently used or to store data and instruction permanently . Depending upon the nature of data stored memory unit are distinguished into primary memory and secondary memory. Memory unit of CPU is a primary memory where data and instruction are stored temporarily. Where as the processed data and information can be stored in secondary memory permanently so that user can retrieve information whenever required. There are two types of memory used in our computer system.

  1. 1. Primary memory/Main memory
  2. 2. Secondary memory/Auxiliary Memory

1) Primary Memory/Main Memory

Primary Memory is also known as main memory of computer. This is the place where our data and instruction are stored before processing and the result are stored before displaying output. Memory which stores data and information currently being used by the computer is known as primary memory.

Features of primary memory:
  • It is not used to store data permanently.
  • It is usually volatile except ROM.
  • It is usually expensive and faster than Secondary storage.
  • It is normally used for smaller storage.
  • It is not used to transfer data from one computer to another computer.
  • It is made up of semiconducting material.

2) Secondary Memory/Auxiliary Memory

Memory which stores data and information permanently for future reference is known as secondary memory. It is a permanent memory where data and instruction remains forever. It is also known as non-volatile memory in which data and instruction remains even if there is no electric supply. This type of memory is also known as auxiliary memory. It has a huge storage capacity up to Gigabytes, Terabytes or even more. Different secondary storage media used are: Magnetic storage, Optical storage and Flash storage.
